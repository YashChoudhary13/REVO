Privacy Policy for REVO (GitHub Explainer)

Last updated: October 7, 2025

1. Introduction

REVO (“we,” “our,” or “us”) respects your privacy. This Privacy Policy describes how we handle information when you use our Chrome extension “REVO – GitHub Explainer.” By using the extension, you agree to this policy.

2. Information We Do Not Collect

REVO does not collect, store, or share any personally identifiable information (PII). We do not request or process data such as names, emails, GitHub credentials, or browsing history.

3. Information We Process

When you use REVO on GitHub, the extension may temporarily process public repository data (e.g., README text, file content, or metadata) for the sole purpose of generating AI-powered summaries and explanations. This data is:

  • Processed in-memory and not stored permanently;
  • Transmitted securely to a remote API (via HTTPS) for analysis;
  • Never used for advertising or tracking purposes.

4. Permissions Used

  • activeTab – Enables REVO to access the content of the active GitHub tab after the user clicks the REVO button.
  • scripting – Allows REVO to inject its overlay UI and dynamically update content on GitHub pages.
  • https://github.com/* – Restricts REVO’s operation to GitHub only.

5. Remote API

REVO uses a remote API to generate AI-based insights. All data sent to this API is non-identifiable text related to the repository content you are viewing. We do not log or retain any personal or private data.

6. Data Security

We employ standard security measures to ensure all data transmission is encrypted and temporary. We do not maintain databases or long-term storage of any processed information.

7. Third-Party Services

Our APIs or hosting providers may process anonymized text strictly for functional purposes (AI explanation generation). They do not access or store personal data, cookies, or GitHub credentials.

8. User Rights

Because we do not collect user data, there is no personal information to access, delete, or export. If you have any concerns, you can still contact us for clarification.
